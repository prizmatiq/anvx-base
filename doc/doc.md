# Синтаксис markdown для базы ANVX

Справка для тех, кто пишет .md-файлы терминов. Обычный markdown (заголовки, списки и т.д.) работает как всегда — здесь только то, что добавлено сверх стандарта.

## Как добавить новый термин

- Скопировать terms/foodservice-erp.html, переименовать под новый термин — внутри ничего не менять
- Рядом положить .md-файл с точно таким же именем
- Добавить ссылку на новую страницу в terms/index.html

## Списки да / нет

:::do
- Foodservice ERP
- ERP-платформа для foodservice
:::

:::do-not
- программа для общепита
- система автоматизации ресторана
:::
